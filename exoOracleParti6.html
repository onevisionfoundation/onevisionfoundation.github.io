Utilisation de la JOINTURE

  1. Ecrire une requête qui affiche les adresses de tous les départements. Afficher les id de location, street address, city, Etat ou province et le pays dans le résultat
  2. Le département HR veut une requête qui affiche le nom, le numéro de département et le nom de département de tous les employés.
  3. Le département HR veut une requête qui affiche tous les employés de Toronto. Afficher le nom, numéro de département, et le nom de département pour tous les employés qui travaille à Toronto
  4. Créer une requête qui affiche le nom et le numéro des employé avec le nom et le numéro de leurs managers. Nommer les colonnes employé », Emp#, Manager et Mgr#, respectivement.
  5. Modifier la requête précédente pour afficher tous les employés y compris King, qui na pas de manager. Ordonner le résulter par le numéro d’employé.
  6. Ecrire une requête qui affiche le nom, le travail, le nom de département, le salarie, et la grade de tous les employé.
  7. Le département HR veut une requête qui affiche le nom et la date d’embauche de tous les employés qui ont été embauché après Davis.
  8. Le département HR veut une requête pour afficher le nom et la date d’embauche de tous les employés embauchés avant leur manager. Afficher aussi le nom let la date d’embauche des manager.

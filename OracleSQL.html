1. Introduction

SQL (Structured Query Language), est un langage destiné à la gestion des bases de données relationnelles. Dans ces tutoriels, nous allons utiliser Oracle comme Système de Gestion de Base de Données (SGBD) dévéloppé par Oracle Corporation. Pour commencer, il faut savoir qu’Oracle comporte plusieurs types d’objets. Les plus utilisés sont les Tables, les Vues, les Triggers (ou Déclencheurs) , les Fonctions, les Procédures et les Packages. Bien sûr il existe d’autres composants dont on va parler dans un autre tutorial.
Oracle corporation
Societé américaine située en Californie, développe et commercialise un SGBD et un ensemble de produits de développement. Oracle a des filiales dans un grand nombre de pays. La filiale française (Oacle France) a été créée en 1986, elle est composée de cinq départements (marketing, comlmercial,avant-vente,conseil et formation).

2. Déblocage du conte HR

Apres intallation d'oracle express 11g, il existe un conte utilisateur HR déja mis en place par la communauté oracle , ce conte permet aux apprenants de gagner du temps dans leurs apprentissages car on y trouve des tableaux déja rempli sur lesquels noux pouvons exécuter de requètes SQL. Mais par défaut, ce conte est bloquer, ce qui nous donne accès aux tableaux.
Dans cette partie, nous allons débloquer le conte HR et lui donner un mots de pass unique nous permettant d'y accedé. Il existe plusieurs methodes pour débloquer le conte HR, dans notre cas, nous allons le débloquer et changer le mot de pass en une seule ligne dans l'invité de commande SQL.
Executer les commandes suivantes:
  1. Connecter vous dabord a votre conte system:
    connect system/[mot_de_pass]
    [mot_de_pass] est le mot de pass fourni lors de l'instalation d'oracle 11g (selon votre version).
  2. Debloquer maintenant le conte HR
    alter user hr account unlock identified by [mot de pass]
    [mot_de_pass] est le mot de pass que vous allez utiliser pour vous connecter au conte HR (à ne pas oublié).
    Vous pouvez vérifié en executant: select * from employees

  1. Ecrivez une requête en SQL pour afficher le nom complet (prénom et nom) et le salaire des employés dont le salaire est inférieur à 6000

  2. Ecrivez une requête en SQL pour afficher le prénom et le nom, le numéro de département et le salaire des employés dont le salaire est supérieur à 8 000
    • select first_name, last_name,department_id,salary from employees where salary > 8000;
  3. Ecrivez une requête en SQL pour afficher le nom, le prénom et le numéro de département de tous les employés dont le nom est "McEwen"
    • select first_name, last_name,department_id,salary from employees where last_name = 'McEwen';
  4. Ecrivez une requête en SQL pour afficher toutes les informations sur le département Marketing

  5. Ecrivez une requête en SQL pour afficher le nom complet (prénom et nom), la date d'embauche, le salaire et le numéro de département des employés dont
    le prénom ne contient pas la lettre M, puis définissez le résultat par ordre croissant de numéro de département.
    • select first_name, last_name,hire_date,department_id,salary from employees where first_name not like '%M%' order by department_id;

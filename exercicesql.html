Exercices Oracle-SQL

  1. Les requetes simples
    1. Ecrivez une requête en SQL pour afficher le nom complet (prénom et nom) et le salaire des employés dont le salaire est inférieur à 6000
      select last_name || first_name nom_complet, salary from employees where salary < 6000;
    2. Ecrivez une requête en SQL pour afficher le prénom et le nom, le numéro de département et le salaire des employés dont le salaire est supérieur à 8 000.
      select first_name,last_name, department_id, salary from employees where salary < 8000;
    3. Ecrivez une requête en SQL pour afficher le nom, le prénom et le numéro de département de tous les employés dont le nom est "McEwen"
      select last_name, first_name, department_id from employees where last_name='McEwen';
    4. Ecrivez une requête en SQL pour afficher toutes les informations sur le département Marketing.
      select * from departments where department_name='Marketing';
    5. Ecrivez une requête en SQL pour afficher le nom complet (prénom et nom), la date d'embauche, le salaire et le numéro de département des employés dont le prénom ne contient pas la lettre M, puis définissez le résultat par ordre croissant de numéro de département.
      select first_name || last_name, hire_date, salary, department_id from employees where first_name not like'%M%' order by department_id;
    6. Ecrivez une requête en SQL pour afficher toutes les informations des employés dont le salaire est compris entre 8 000 et 12 000 et dont la commission n’est pas nulle ou le numéro de département est différent des numéros 40, 120 et 70 et qui ont été embauchés avant le 5 juin. 1987.
      select * from employees where salary between 8000 and 12000 and commission_pct is not null or department_id not in (40,120,70) and hire_date < '05/06/1987';
    7. Ecrivez une requête en SQL pour afficher le nom complet (prénom et nom) et le salaire de tous les employés qui ne gagnent aucune commission.
      select last_name || first_name nom_complet,salary from employees where commission_pct is null;
    8. Ecrivez une requête en SQL pour afficher le nom complet (prénom et nom), le numéro de téléphone et l'adresse électronique, séparés par un trait d'union et le salaire, pour les employés dont le salaire est compris entre 9 000 et 17 000. Les en-têtes de colonne affectés à Full_Name , Contact_Details et Rémuity, respectivement.
      select last_name ||first_name Full_Name, phone_number || '_' || email contact_details, salary salary from employees where salary between 9000 and 17000;
    9. Ecrivez une requête en SQL pour afficher le prénom, le nom de famille et le salaire des employés dont le prénom se termine par la lettre m.
      select first_name, last_name, salary from employees where first_name like'%m';
    10. Ecrivez une requête en SQL pour afficher le nom complet (prénom et nom), ainsi que le salaire, pour tous les employés dont le salaire est compris entre 7000 et 15000, et définissez le résultat dans l'ordre croissant du nom complet.
      select last_name || first_name nom_complet, salary from employees where salary between 7000 and 15000 order by nom_complet ;
    11. Ecrivez une requête en SQL pour afficher le nom complet (prénom et nom), l'identifiant du travail et la date d'embauche des employés embauchés le 5 novembre 2007 et le 5 juillet 2009.
      select last_name, first_name, job_id, hire_date from employees where hire_date in('05/11/2007','05/07/2009');
    12. Ecrivez une requête en SQL pour afficher le nom complet (prénom et nom) et le numéro de département des employés travaillant dans les départements 70 ou 90.
      SELECT last_name, first_name, department_id from employees where department_id in (70,90);
    13. Ecrivez une requête en SQL pour afficher le nom complet (prénom et nom), le salaire et le numéro du responsable pour les employés travaillant sous un responsable.
      select last_name, first_name, salary, manager_id from employees where manager_id is not null;
    14. Ecrivez une requête en SQL pour afficher toutes les informations de la table Employés concernant les employés embauchés avant le 21 juin 2002.
      select * from employees where hire_date < '21/06/2002';
    15. Ecrivez une requête en langage SQL pour afficher le nom et le prénom, l'adresse électronique, le salaire et l'ID du responsable, pour les employés dont le responsable est titulaire des identifiants 120, 103 ou 145.
      select last_name, first_name, email, salary, manager_id from employees where manager_id in(120,103,145);
    16. Ecrivez une requête en SQL pour afficher toutes les informations concernant tous les employés portant les lettres D, S ou N en leur prénom, puis organisez le résultat en ordre décroissant de salaire.
      select * from employees where first_name like'%D%' or first_name like'%S%' or first_name like'%N%' order by salary desc;
    17. Ecrivez une requête en SQL pour afficher le nom complet (prénom et nom), la date d'embauche, le pourcentage de commission, l'adresse e-mail et le téléphone séparés par '-', ainsi que le salaire des employés dont le salaire est supérieur à 11 000 ou au septième chiffre. dans leur numéro de téléphone est égal à 3 et faire le résultat défini dans un ordre décroissant par le prénom.
      select last_name ||' '|| first_name, hire_date, commission_pct*100/(select sum(commission_pct) from employees) as commission, email ||'-'|| phone_number , salary from employees where salary > 11000 or substr(phone_number,7,1)='3' order by first_name desc;
    18. Ecrivez une requête en SQL pour afficher le prénom et le nom, ainsi que le numéro de département des employés dont la lettre s est le 3ème caractère de leur prénom.
      select first_name, last_name, department_id from employees where substr(first_name,3,1)='s';
    19. Écrivez une requête en SQL pour afficher l'ID, le prénom, l'ID de travail et le numéro de service de l'employé qui travaille sauf les départements 50, 30 et 80.
      select employee_id, first_name, job_id, department_id from employees where department_id not in(50,30,80);
    20. Ecrivez une requête en SQL pour afficher l'ID, le prénom, l'ID de travail et le numéro de service de l'employé dont le numéro de service est égal à 30, 40 ou 90.
      select employee_id, first_name, job_id, department_id from employees where department_id in(30,40,90);
    21. Ecrivez une requête en SQL pour afficher le prénom, le nom de famille et la date d'adhésion des employés, qu'ils soient représentant commercial ou commercial.
    22. Ecrivez une requête en SQL pour afficher l'ID d'employé et la date à laquelle il a terminé son travail précédent.
    23. Ecrivez une requête en SQL pour afficher les détails des employés qui n’ont pas de pourcentage de commission ni de salaire compris entre 7 000 et 12 000 et qui travaillent dans ce département dont le nombre est 50.
    24. Ecrivez une requête en SQL pour afficher tous les employés dont le prénom ou le nom commence par la lettre D.
    25. Ecrivez une requête en SQL pour afficher les employés qui ont rejoint le groupe après le 7 septembre 1987.
